# Python - Async Comprehension

This directory holds all python scripts and files generated during resolution of the assigned tasks by ALX Africa on the topic of usage of Asynchronous Comprehension in Python.

## Requirements

### General

	->	A README.md file, at the root of the folder of the project, is mandatory
	->	Allowed editors:
			* vi,
			* vim,
			* emacs
	->	All files will be interpreted/compiled on:
			Ubuntu 18.04 LTS
				using:
					python3 (version 3.7)
	->	All files should end with a new line
	->	All files must be executable
	->	The length of files will be tested using:
			* wc
	->	The first line of all files should be exactly:
			#!/usr/bin/env python3
	->	All code should use:
			* The pycodestyle style (version 2.5.x)
	->	All functions and coroutines must be type-annotated.
	->	All modules should have a documentation:
			python3 -c 'print(__import__("my_module").__doc__)'
	->	All functions should have a documentation:
			python3 -c 'print(__import__("my_module").my_function.__doc__)'
	->	A documentation is not a simple word, it’s a real sentence explaining what’s the purpose of the module, class or method
